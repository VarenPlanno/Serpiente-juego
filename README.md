# Proyecto: Prototipo de Snake (De Scratch a Pytch)

Este proyecto consiste en el desarrollo de un prototipo interactivo del clásico juego de la serpiente (*Snake*). El objetivo principal de la entrega es experimentar con la migración de lógicas visuales de bloques hacia la programación basada en texto, utilizando la plataforma **Pytch** como puente de desarrollo.

## Características del Juego
* **Movimiento Continuo:** El personaje principal se desplaza por el lienzo respondiendo a los comandos del teclado.
* **Mecánica de Interacción:** El objetivo consiste en recolectar elementos simples esparcidos por la pantalla para sumar puntos.
* **Reajuste de Lógica:** Adaptación de las estructuras de control, bucles y variables originalmente planteados en Scratch hacia la sintaxis de Python en Pytch.

## Controles
* **Flechas de dirección (Arriba, Abajo, Izquierda, Derecha):** Controlan el rumbo de la serpiente por el escenario.
* **Tecla Espacio/ Bandera Verde:** Iniciar / Reiniciar la partida.

## Tecnologías Utilizadas
* **Scratch:** Utilizado para la conceptualización inicial y el testeo rápido de la lógica de bloques.
* **Pytch:** Entorno web para la traducción de la lógica de bloques a código Python.
